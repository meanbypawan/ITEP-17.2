1. Python Introduction (Interview Qs)
❓ What is Python?

👉 Python is a high-level, interpreted, dynamically typed, object-oriented programming language.

❓ Why is Python called interpreted?

👉 Code is executed line-by-line by the Python Virtual Machine (PVM) after compilation to bytecode.

❓ Is Python compiled or interpreted?

👉 Both:

Compiled → .pyc (bytecode)
Interpreted → executed by PVM

❓ What are key features of Python?
Easy syntax
Dynamically typed
Cross-platform
Large libraries
Supports OOP & functional programming

❓ What is PVM?

👉 Python Virtual Machine executes bytecode.
