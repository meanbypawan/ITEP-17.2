HTML is all about Tag[Tag-based]
1. Paired   <opening></closing>
    <p>Hello.......</p> 

2. Unpaired/Self-closing
   <br>, <input/>, <img/>

3. Structural Tag
   html, body,head,title,meta.....

4. Semantics Tag/SEO
   header,footer,main,section,article.....

5. Heading Tag
   h1,h2,h3,h4,h5,h6

6. Listing
    ordered list :--- ol
    unordered list :-- ul
    list-item : li

7. Text Formatting
   b, i, u, strong, del, sup, sub 
8. Media
   img, audio, vedio, picture, Object
9. Form related Tag
   form, input, button
   <input type=""/>
   ["text","email","passsord","number", "date","color","file",
   "radio","checkbox","button"]
   select
10. Table
    table,thead,tfoot,tr,th,td
11. Inline Tag :-
    label, span, small...
12. Block Level-Tag    
    div,p,h1...h6.
